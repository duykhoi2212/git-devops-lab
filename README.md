# DevOps Midterm - git-devops-lab

## Stack
- Backend: Node.js + Express
- Frontend: HTML + Nginx  
- Database: MongoDB

## Run
```bash
docker compose up -d
```
- Frontend: http://localhost
- Health: http://localhost:3000/health
- About: http://localhost:3000/about
